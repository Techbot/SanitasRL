README

SanitasRL was written in Javascript by Codejunkie for the 7DRL challenge. No Roguelike libraries where used and everything was written from scratch during the 7 days of the challenge. You can read my posts about the development on 7drl.org here.

You can play the game in your browser at http://codejunkie.se/sanitasrl/ or download the game and open index.html in your browser. It requires a modern browser which supports the <canvas> element.

When you play the game on this page or download it the source will be compressed. If you wish to look at the source code, please don't it's horrible, you may download the uncompressed version here.

This game is in the public domain, you may do whatever you choose with it.
